LOWESS is a simple, efficient, general-purpose non-parametric smoothing algorithm. It perform weighted linear regressions on subsets of neighboring points, yielding a smooth curve fit to the data.

Template Parameters
Data_Floating-point type for the data.
Parameters
num_pointsNumber of points.
xPointer to an array of num_points x-values. This should be sorted in increasing order, see SortBy for details.
yPointer to an array of num_points y-values.
[out]fittedPointer to an output array of length n, in which the fitted values of the smoother can be stored.
[out]robust_weightsPointer to an output array of length n, in which the robustness weights can be stored. This may be NULL if the robustness weights are not needed.
optFurther options.
